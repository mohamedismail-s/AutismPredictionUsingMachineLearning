## 📖 Overview

Early detection of **Autism Spectrum Disorder (ASD)** can significantly improve intervention outcomes. This project explores the application of Machine Learning techniques to predict ASD by analyzing screening questionnaire data.

The repository presents a comparative study of multiple classification algorithms, feature selection techniques, and evaluation metrics to determine the most effective model for ASD prediction.

---

## ✨ Key Features

* 📊 Data preprocessing and cleaning
* 🔍 Feature Selection using SelectKBest (Chi-Square)
* 🤖 Multiple Machine Learning models
* 📈 Performance comparison of classifiers
* 📉 Confusion Matrix and Classification Report
* 📊 ROC Curve & Evaluation Metrics
* 📓 Well-documented Jupyter Notebooks
* 📄 Complete research documentation

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/mohamedismail-s/AutismPredictionUsingMachineLearning.git
```

### Navigate to the project

```bash
cd AutismPredictionUsingMachineLearning
```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn scikit-learn xgboost lightgbm catboost notebook
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook and run the cells sequentially.
